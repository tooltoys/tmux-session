# Tmux workspace

Tạo tab tmux, rename với json config.

```json
[
  {
    "path": "/home/ubuntu/code/learning/dsa/algorithm",
    "name": "algo"
  },
  {
    "path": "/home/ubuntu/code/learning/backend/golang",
    "name": "go"
  }
]
```

Trong đó 

+ path: là đường dẫn đến dir cần thao tác.
+ name: custom name cho tmux panel.
